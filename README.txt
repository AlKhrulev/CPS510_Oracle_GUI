Requirements:
Python 3.x (preferably 3.8)
Oracle_cx
Tkinter

How to use GUI:
Open the executable file
Open config.py file and change the variables to the correct connection details for Ryerson Oracle Database
Establish connection by pressing the button
Create and populate tables by pressing the buttons
Use the query button to submit custom queries
You will be able to see the results in a new pop-up window
Press drop to erase created tables

Clarification: Queries do not have semicolons in our text files for formatting due to our program
